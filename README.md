## what does it do

- throws confetti at your screen when you merge a PR
- 4 levels of confetti chaos to choose from

## installation

1. download or clone this repository
2. open Chrome and go to `chrome://extensions/`
3. turn on "Developer mode" using the toggle in the top right
4. click "Load unpacked"
5. select the `bitbucket-confetti` folder
6. done - the extension icon should appear in your toolbar

## how to use it

1. open a PR on Bitbucket
2. merge it
3. confetti happens
4. feel good about yourself

### settings

click the extension icon to mess with stuff:

- pick your chaos level
- test confetti
